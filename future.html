Chapter 4

Designing a Future Where Sport Meets Tech

I see a future where Kenyan girls lace up skates with the same ease they open their favorite learning app. My work will blend UX research, sport innovation, and community storytelling to build that reality.

Short-term (1‑2 years)

  • Finish my Zetech thesis on wearable Roll Ball telemetry.
  • Coach Meru county girls to qualify for nationals.
  • Publish two bilingual case studies on inclusive sport design.

Mid-term (3‑5 years)

  • Pursue graduate studies in Human-Computer Interaction.
  • Co-found a sports-tech studio for African women innovators.
  • Release an open-source API for Kenyan roll ball statistics.

Domain & Hosting Strategy

Domain Name

I plan to register winnermutembei.com through Namecheap so the domain mirrors my identity and is easy to spell globally.

Configuration: enable WHOIS privacy, enforce HTTPS via SSL, and set renewal reminders.

Web Hosting

I deploy via GitHub Pages (version controlled) and layer Cloudflare for CDN caching, DDoS mitigation, and automatic TLS before mapping the custom domain.

Security & Maintenance Plans

  • Use GitHub Dependabot and secret scanning on the repo.
  • Run automated accessibility + Lighthouse audits pre-deploy.
  • Review content quarterly, including roll ball stats embeds.

Impact Roadmap

  • Mentor 100 girls via annual skate-and-code bootcamps.
  • Create policy briefs advocating for safe city skate lanes.
  • Translate UX findings into Swahili and Kimeru toolkits.

Long-term Vision

Build an East African movement lab that pairs urban planners, skaters, and technologists to prototype inclusive recreation corridors.

Every page in this site is both a reflection and a blueprint toward that future.
